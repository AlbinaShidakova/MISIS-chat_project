В этом файле описаны команды для общения между клинтом и сервером.

Все команды от клиента к серверу должны поступать в формате json.
у каждой команды может быть свой список атрибутов (См. пример).
Пример команды:
    {'cmd':'VK_Autorization', 'username' : "user1", 'password':"pass1"}

Поддерживаемые команды:
    VK_Autorization - авторизация пользователя через ВК:
        Аттрибуты:
            username - Имя пользователя в ВК
            password - пароль в ВК.
        Ответ:
            {"Error":"0"/"1", "Answer" : "Incorrect username or password"/"Autorization OK")
    Login - авторизация в чате:
        Аттрибуты:
        Ответ:
    Registration - регистрация в чате (заполнение анкеты):
